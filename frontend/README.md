# FreelanceFlow Frontend

## Stack
- React.js
- Tailwind CSS
- Recharts (charts)
- React Router
- JWT Auth (token-based)

## Setup
1. `npm install`
2. `npm start` to run the app

## Features
- Dashboard: Active Projects, Pending Invoices, Upcoming Deadlines
- CRUD: Clients, Projects, Tasks
- Time Tracking: Stopwatch & Manual Entry
- Invoicing: PDF download
- Financial Charts
- Sample Data Loader

## Notes
- Connects to backend API (see backend/README.md)
- Timer state persists via localStorage
