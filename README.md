# FreelanceFlow

A B2B SaaS dashboard for freelancers to manage clients, projects, tasks, time tracking, and invoicing in one place.

## Tech Stack
- Backend: Node.js, Express.js, PostgreSQL
- Frontend: React.js, Tailwind CSS
- Auth: JWT
- PDF Generation: PDFKit (backend)
- Charts: Recharts (frontend)

## Features
- Client, Project, Task CRM
- Time Tracking (real-time & manual)
- Invoice Generation (PDF)
- Financial Dashboard (charts)
- Multi-Tenancy & Tiered Access
- Sample Data Loader

## Getting Started
1. Clone the repo
2. See backend/ and frontend/ folders for setup instructions
3. Load sample data from the dashboard or API

## Folder Structure
- backend/ — Express API & DB
- frontend/ — React app
